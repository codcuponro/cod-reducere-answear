# Cod Reducere Answear

O colecție de coduri de reducere Answear. Le folosim pentru testarea cuvintelor cheie cod reducere Answear, voucher Answear, cupon Answear, și cod promotional Answear de pe Cuponescu.ro.

## Instalare

Instalează `cod-reducere-answear` prin NPM:

```bash
npm install cod-reducere-answear
```

## Utilizare

Pachetul conține un array de obiecte reprezentând coduri de reducere.

În Node:

```javascript
var codes = require('cod-reducere-answear')

console.log(codes)

// [
//   {
//     site: 'https://www.answear.ro',
//     cod_reducere: 'ANSWEAR10',
//     reducere: '10% reducere',
//     descriere: 'Reducere de 10% la toate produsele'
//   },
//   ...
// ]
```
